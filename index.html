Abstract

Netflix is one of the biggest streaming platforms with a large collection of movies and TV shows. Understanding trends in content helps viewers, producers, and marketers. This project analyzes a Netflix dataset using Pandas Profiling. The goal is to find patterns in genre, release year, ratings, and content type. This helps in understanding what kind of content is popular.

Introduction

Netflix has thousands of movies and TV shows from different countries and genres. Users choose content based on ratings, genre, and popularity. Analyzing Netflix data helps us see trends like which genres are most common, which countries produce the most content, and how ratings vary. In this project, we use Pandas Profiling to explore a Netflix dataset and find useful insights

Literature Survey

Many researchers have studied streaming platform trends using data analysis. Some use machine learning to recommend movies based on user preferences. Others study content trends to understand what type of movies and TV shows get high ratings. Traditional analysis is done through surveys, but automated analysis with tools like Pandas Profiling provides better and faster insights. Studies show that Exploratory Data Analysis (EDA) is useful for analyzing
entertainment industry trends.

EDA

EDA helps in understanding trends in Netflix content. Using Pandas Profiling, we can find patterns in content type (movies vs. TV shows), most common genres, top-rated content, and release year trends. We can also see which countries produce the most Netflix content and how ratings are distributed.
